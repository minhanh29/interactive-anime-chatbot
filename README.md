# Serena Chatbot

## Installation
1. Install packages
```bash
pip install -r requirements.txt
```
2. Run database in Docker
```bash
docker compose up
```
3. Start the API
```bash
python api.py
```
4. Start the UI
```bash
python ui.py
```
5. Enable continual knowledge updating. [Optional]
```bash
python ./mutation_thread.py
```
